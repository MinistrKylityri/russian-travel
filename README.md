# Путешествия по России
Данный проект написан с целью показать возможности HTML и CSS в адаптивной верстке с применением таких технологий, как:
1. flex-контейнер.
2. Анимация.
3. Grid.
4. БЭМ.
5. Медиа запросы.

Страница выполнена на русском языке.
